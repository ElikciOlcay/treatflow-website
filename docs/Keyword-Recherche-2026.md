## 6. Keywords mit externem Tool prüfen (Suchvolumen)

Die Empfehlungen oben basieren auf Suchintention und Wettbewerb. **Suchvolumen** (monatliche Suchanfragen in DE) kannst du selbst mit diesen Tools prüfen:

### Option 1: SearchVolume.io (kostenlos, Bulk)

1. Auf **https://searchvolume.io/** gehen.
2. Land auf **Germany** stellen.
3. Keywords einfügen (ein Begriff pro Zeile), z. B.:
   - Kosmetikstudio Software  
   - Software für Kosmetikstudio  
   - Kundenkartei Kosmetik  
   - digitale Kundenkartei Kosmetik  
   - Terminplaner App Kosmetikstudio  
   - Online Terminbuchung Kosmetik  
   - Terminbuchungssoftware Kosmetik  
   - NiSV Dokumentation Software  
   - Anamnese Software Kosmetik  
   - Beste Kosmetikstudio Software  
   - Behandlungsdokumentation Software Kosmetik  
   - Beauty Salon Software  
4. Auf **Submit** klicken – die Tabelle zeigt Suchvolumen pro Keyword.
5. Keywords mit hohem Volumen und passender Intention priorisieren; niedriges Volumen ist bei Nischen-Keywords normal und kann trotzdem lohnend sein.

### Option 2: Google Keyword Planner (kostenlos mit Google-Konto)

1. **https://ads.google.com** – Google Ads-Konto anlegen („Konto ohne Kampagne erstellen“ reicht).
2. Unter **Tools & Einstellungen** → **Keyword-Planer** → **Keyword-Ideen ermitteln**.
3. Suchbegriffe oder eure Website-URL eingeben.
4. **Standort:** Deutschland (und ggf. Österreich).
5. Suchvolumen und Wettbewerbsgrad werden als Bereiche angezeigt (Google zeigt keine exakten Zahlen mehr ohne aktive Kampagne, aber die Reihenfolge hilft bei der Priorisierung).

### Option 3: Bezahlte Tools (genaue Zahlen + Schwierigkeit)

- **Ahrefs** (Keywords Explorer), **SEMrush** (Keyword Magic Tool), **Sistrix** (DE-fokussiert): exaktes Suchvolumen, Keyword Difficulty, CPC. Ideal für regelmäßige Keyword-Planung.
- **Ubersuggest** (Neil Patel): begrenzte kostenlose Abfragen, danach Abo.
